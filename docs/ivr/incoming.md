# IVR呼入回调（尚未开放）
<!-- toc -->

该接口由开发者实现，壹云服务器在监听到ivr呼入事件时，将通过该接口询问开发者是否接听该呼入。
开发者可根据呼入号码进入必要的判断，比如是否黑名单


##回调URL
    
```
GET ${NOTIFY_URL}?from={mobile}
```

### 参数列表
| 参数                  | 有效值范围  |  说明                                     |
| --------------------- | ----------- | ---------------------------------------- |
| `action`            | **ivr_incoming** |询问开发者是否接听呼入标志：ivr_incoming |
| `from`                 | 号码字符串  |  * 呼入号码                                 |


##响应
响应文本`accept`表示接收

其它表示拒接
