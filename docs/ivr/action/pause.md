# 暂停（尚未开放）

## pause 节点

```
pause
```

## 属性

| 参数                  | 说明                                      |
| --------------------- |  ---------------------------------------- |
| `duration_keys`          | 暂停时长（秒）                  |


## 内容

无

## 嵌套

`next`

## 示例

```xml
<response>
    <pause duration="30"></pause>
    <next>http://yourhost/nextstep</next>
</response>
```
